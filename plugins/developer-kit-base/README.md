# Developer Kit - Base

Universal development toolkit with language-agnostic skills for git workflow, documentation, and README generation.

## Skills (3)

| Skill | Description |
|-------|-------------|
| `create-readme` | README generation with proper structure, badges, and installation instructions |
| `documentation-writer` | Documentation writing following the Diátaxis framework |
| `git-commit` | Git commit with Conventional Commits specification |

## Installation

```bash
make install
```
